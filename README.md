# Twitter API Test

## Usage

### Get API KEY

https://rapidapi.com/omarmhaimdat/api/twitter154

### Set environment variables

```shell
cp .env.example .env
```

### Install Package

```shell
pip install -r requirements.txt
```

### Run the code

#### Get followers
    
```shell
python3 followers.py
```

#### Search retweets
    
```shell
python3 search_retweets.py
```
